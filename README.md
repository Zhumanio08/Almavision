### 1. О проекте

Almavision — это веб-дашборд для интеллектуального управления городом на основе данных. Система предназначена для мониторинга и анализа ключевых показателей транспортной и экологической ситуации в 8 районах города Алматы.

Проект предоставляет лицам, принимающим решения (мэр, руководители управлений, главы районов), инструмент для оперативного получения актуальной статистики, выявления аномалий и получения рекомендаций от ИИ-ассистента.
Основные возможности:

    📊 Дашборд районов — карточки всех 8 районов с краткой статистикой при наведении

    🏙️ Сводная статистика города — агрегированные данные по всем районам

    📈 Интерактивные графики — визуализация динамики показателей по часам (с утренним и вечерним пиком)

    📋 KPI-таблицы — цветовая индикация статуса каждого показателя (зелёный/жёлтый/красный)

    🤖 AI-советник — интеграция с Groq API (Llama 3.3 70B) для генерации управленческих рекомендаций

    🔄 Обновление данных — генерация новых случайных данных с сохранением реалистичных пиковых нагрузок

    🎯 Сортировка районов — по приоритету, загруженности или AQI
